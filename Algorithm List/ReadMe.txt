Description -

名字 - MyLinkList

用途 - 帮你将数据组织成链表

实用类型 - 基本数据类型/自定义数据类型

帮助 -

（1）需要使用 MyLinkList 时，将头文件 list.h 包含进你的项目即可

（2）提供的操作：

a.无参构造函数 
b.拷贝构造函数
c.初始化链表长度的构造函数
d.重载 = 号操作（拷贝链表）
e.析构函数
f.插入节点
g.删除节点
h.连接两个链表
i.重置链表
j.返回头节点地址
k.返回尾节点位置
l.重新设置长度（将删除原有链表）
m.重载下标操作符（以访问链表元素）

（3）.如果要使用 = 号操作符对链表进行拷贝，请用户在自定义类中遵循 “三原则”（即定义 复制构造函数/ 重载 = 号/ 析构函数）

PS：如果使用的时内定数据类型，无需考虑 帮助（3）
